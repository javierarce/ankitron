## Global
| Shortcut | Action |
| --- | --- |
| <kbd>Cmd</kbd> + <kbd>K</kbd> | Open the command palette |
| <kbd>Cmd</kbd> + <kbd>N</kbd> | New note |
| <kbd>Cmd</kbd> + <kbd>,</kbd> | Open settings |
| <kbd>Cmd</kbd> + <kbd>1</kbd> | Go to Study |
| <kbd>Cmd</kbd> + <kbd>2</kbd> | Go to Decks |

## Command palette 
| Shortcut | Action |
| --- | --- |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move the selection |
| <kbd>Enter</kbd> | Open the selected deck / run the action |
| <kbd>Esc</kbd> | Go back a level, or close the palette |

## List navigation 
| Shortcut | Action |
| --- | --- |
| <kbd>j</kbd> / <kbd>↓</kbd> | Move down |
| <kbd>k</kbd> / <kbd>↑</kbd> | Move up |
| <kbd>g</kbd> <kbd>g</kbd> | Jump to the top (press <kbd>g</kbd> twice within 500 ms) |
| <kbd>G</kbd> (<kbd>Shift</kbd> + <kbd>g</kbd>) | Jump to the bottom |
| <kbd>l</kbd> / <kbd>→</kbd> | Expand the focused deck (in the deck list) |
| <kbd>h</kbd> / <kbd>←</kbd> | Collapse the focused deck, or jump to its parent (in the deck list) |

## Decks page
| Shortcut | Action |
| --- | --- |
| <kbd>Cmd</kbd> + <kbd>F</kbd> or <kbd>/</kbd> | Focus the search box |

## Note list (deck detail)
| Shortcut | Action |
| --- | --- |
| <kbd>Cmd</kbd> + <kbd>F</kbd> or <kbd>/</kbd> | Focus the search box |
| <kbd>Cmd</kbd> + <kbd>A</kbd> | Select all notes |
| <kbd>Space</kbd> | Toggle selection of the focused note |
| <kbd>Shift</kbd> + <kbd>J</kbd> / <kbd>Shift</kbd> + <kbd>K</kbd> | Move down/up, extending the selection |
| <kbd>a</kbd> | Add a new note |
| <kbd>e</kbd> | Edit the selected notes in sequence (or the focused note if none are selected) |
| <kbd>t</kbd> | Add or remove tags on the selected notes (or the focused note if none are selected) |
| <kbd>Cmd</kbd> + <kbd>Z</kbd> | Undo the last tag change |

## Study session
| Shortcut | Action |
| --- | --- |
| <kbd>Space</kbd>, <kbd>1</kbd>, or <kbd>2</kbd> | Reveal the answer |
| <kbd>1</kbd> | Grade **Fail** (after reveal) |
| <kbd>Space</kbd> or <kbd>2</kbd> | Grade **Pass** (after reveal) |
| <kbd>r</kbd> | Play the card's audio (answer side once revealed, otherwise the question) |
| <kbd>e</kbd> | Edit the current note |
| <kbd>a</kbd> | Add a note to the session |
| <kbd>Cmd</kbd> + <kbd>←</kbd> | Return to the deck (confirms once you've started reviewing) |
| <kbd>Cmd</kbd> + <kbd>Z</kbd> | Undo the last review |
