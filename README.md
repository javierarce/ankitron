# AnkiTron

A mac app for managing and studying [Anki](https://apps.ankiweb.net) decks. Browse decks, create and edit cards, and run study sessions in a simple and clean interface.

## Features

- Browse all decks with due-card counts
- Create, edit, and delete cards (Basic and Cloze note types)
- See and manage card tags
- JSON import/export — edit decks offline and re-import them to update existing notes or add new ones
- Undo the last review with `z` (or `Cmd`/`Ctrl-Z`) during study
- Light/dark theme toggle with system preference detection

## Prerequisites

1. [Anki desktop](https://apps.ankiweb.net).
2. The [AnkiConnect](https://ankiweb.net/shared/info/2055492159) add-on must be installed.

## Install (macOS)

1. Download the latest `AnkiTron_*_universal.dmg` from the [Releases page](https://github.com/javierarce/ankitron/releases). 
2. Mount the DMG, drag AnkiTron to **Applications**, and launch it.
